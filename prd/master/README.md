# Master PRD Index

This folder contains the master product requirements for POD OS.

## Documents

- `01-overview.md` — product objective, users, and main workflow.
- `02-core-modules.md` — core product modules.
- `03-operational-rules.md` — billing, production, shipping, returns, and acceptance rules.

## Build Principle

The platform must be built as a complete print fulfillment operating system. It must include seller software, admin software, production software, warehouse software, store connectors, billing, shipping, returns, inventory, analytics, and audit logging.

## Important Rule

All PRD documents must stay inside the top-level `prd/` folder.
